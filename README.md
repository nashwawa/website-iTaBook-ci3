# Website iTaBooks

## Deskripsi

Website Toko Buku Online merupakan aplikasi berbasis **CodeIgniter 3** yang dirancang untuk memudahkan pengguna dalam mencari, melihat, dan membeli buku secara online. Aplikasi ini dilengkapi dengan fitur pengelolaan data buku, kategori, keranjang belanja, transaksi, serta konfirmasi pembayaran sehingga proses jual beli menjadi lebih praktis dan terorganisir.

---

## Fitur Utama

* Login Admin
* Login & Registrasi Pengguna
* Dashboard Admin
* Manajemen Data Buku
* Manajemen Kategori Buku
* Manajemen Data Pengguna
* Pencarian Buku
* Detail Buku
* Keranjang Belanja
* Checkout
* Upload Bukti Pembayaran
* Riwayat Pesanan
* Konfirmasi Pesanan
* Laporan Penjualan
* Logout

---

## Teknologi yang Digunakan

* PHP
* CodeIgniter 3
* MySQL
* Bootstrap
* HTML
* CSS
* JavaScript

---

## Cara Menjalankan Project

1. Clone atau download repository ini.
2. Pindahkan project ke folder **htdocs** (XAMPP).
3. Import database yang terdapat pada folder **database**.
4. Atur koneksi database pada file `application/config/database.php`.
5. Jalankan Apache dan MySQL melalui XAMPP.
6. Buka project melalui browser.
